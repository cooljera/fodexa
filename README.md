# FODEXA

Sistema de gestión para restaurantes. Este repositorio incluye módulos en JavaScript puro y algunos componentes en React para la sección de ventas.

## 🛠 Construir componentes de ventas

Si se modifica cualquier archivo dentro de `src/components/ventas`, ejecute:

```bash
npm install     # solo la primera vez
npm run build
```

Los archivos compilados se guardarán en `dist/components/ventas` y pueden incluirse directamente en las páginas HTML.

Para detalles consulte `docs/COMPONENTES_VENTAS_BUILD.md`.
