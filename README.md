# FODEXA

FODEXA is a modular restaurant management system built entirely with HTML, CSS and vanilla JavaScript. It stores data locally using JSON files, localStorage and IndexedDB. No Python or Node backend is required.

Use the provided batch scripts to open the files directly or start a simple local server (e.g. `python -m http.server 8000`) to avoid browser security restrictions when loading JSON files.

All business modules live under the `modulos/` directory and each module contains its own HTML, JS and CSS files.
